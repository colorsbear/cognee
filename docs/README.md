## 📖 核心概念速记

### Cognee 工作流

```
add() → cognify() → search()
  ↓         ↓           ↓
添加数据   构建图谱    查询知识
```

### 核心组件

| 组件 | 位置 | 作用 |
|------|------|------|
| API 层 | `cognee/api/v1/` | HTTP 接口和 SDK 入口 |
| 流水线 | `cognee/modules/pipelines/` | 任务编排引擎 |
| 任务 | `cognee/tasks/` | 具体业务逻辑实现 |
| 检索器 | `cognee/modules/retrieval/` | 搜索实现 |
| 基础设施 | `cognee/infrastructure/` | 数据库和 LLM 适配器 |

### 数据库层次

```
关系数据库 (SQLite/PostgreSQL) - 元数据和用户数据
    ├── 图数据库 (Kuzu/Neo4j) - 实体和关系
    └── 向量数据库 (LanceDB/ChromaDB) - 语义搜索
```
