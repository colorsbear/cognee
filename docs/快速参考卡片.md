# Cognee 快速参考卡片

## 一、核心 API

```python
import cognee
import asyncio

async def main():
    # 1. 添加数据
    await cognee.add("Your text or file path", dataset_name="my_dataset")
    
    # 2. 构建知识图谱
    await cognee.cognify(datasets=["my_dataset"])
    
    # 3. 搜索
    results = await cognee.search("What is this about?")
    
    # 4. 删除
    await cognee.delete(data_id=some_id)

asyncio.run(main())
```

## 二、搜索类型速查

```python
from cognee import SearchType

# 图谱完成（默认，最智能）
await cognee.search("复杂问题", query_type=SearchType.GRAPH_COMPLETION)

# 文档块检索（最快）
await cognee.search("查找特定段落", query_type=SearchType.CHUNKS)

# 传统 RAG
await cognee.search("文档检索", query_type=SearchType.RAG_COMPLETION)

# 摘要检索
await cognee.search("快速概览", query_type=SearchType.SUMMARIES)

# 代码搜索
await cognee.search("查找函数", query_type=SearchType.CODE)

# 智能选择
await cognee.search("任意问题", query_type=SearchType.FEELING_LUCKY)
```

## 六、常见问题一键解决

```bash
# 问题：LLM 连接超时
# 解决：跳过连接测试
export COGNEE_SKIP_CONNECTION_TEST=true

# 问题：图谱为空
# 解决：运行 cognify
await cognee.cognify(datasets=["my_dataset"])

# 问题：权限错误
# 解决：禁用多租户
export ENABLE_BACKEND_ACCESS_CONTROL=false

# 问题：内存不足
# 解决：减小批量大小
await cognee.cognify(chunks_per_batch=10, data_per_batch=5)
```

## 九、文件路径格式

```python
# 正确的路径格式
await cognee.add("/absolute/path/to/file.pdf")          # ✓ 绝对路径
await cognee.add("file://relative/path.txt")            # ✓ 文件 URL
await cognee.add("file:///absolute/path.docx")          # ✓ 绝对文件 URL
await cognee.add("s3://bucket/file.pdf")                # ✓ S3 路径

# Windows 路径
await cognee.add(r"D:\data\file.pdf")                   # ✓ 原始字符串
await cognee.add("D:/data/file.pdf")                    # ✓ 正斜杠
```

## 十、性能调优参数

```python
await cognee.cognify(
    datasets=["large_dataset"],
    chunks_per_batch=50,        # 每批处理的 chunk 数（默认 100）
    data_per_batch=10,          # 每批处理的数据项数（默认 20）
    chunk_size=512,             # 每个 chunk 的 token 数（自动计算最优）
    run_in_background=True,     # 后台运行（大文件推荐）
)
```

## 十一、自定义图谱模型

```python
from cognee.infrastructure.engine import DataPoint

class ScientificPaper(DataPoint):
    title: str
    authors: list[str]
    abstract: str
    year: int
    
    class Metadata:
        index_fields = ["title", "abstract"]

await cognee.cognify(
    datasets=["papers"],
    graph_model=ScientificPaper
)
```
